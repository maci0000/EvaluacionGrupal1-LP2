GitHub Pages

¿Qué es?

Servicio gratuito de GitHub para alojar sitios web estáticos directamente desde un repositorio.

¿Cómo configurarlo?

  1. Ir a Settings > Pages
  2. Seleccionar la rama (main o gh-pages)
  3. Especificar la carpeta (/docs o /src)
  4. Guardar los cambios

El sitio estará disponible en: https://[usuario].github.io/[repositorio]
